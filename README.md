# 📍 Unified Location Tracker using Google Maps JavaScript API

This project demonstrates real-time user location tracking on an interactive map using the browser Geolocation API and Google Maps JavaScript API. The application continuously updates the user's position and dynamically centers the map.

👉 Click to watch the demo video:

## 🚀 Features

- 📡 Live location tracking using Geolocation API
- 🗺️ Interactive Google Map integration
- 📍 Automatic marker updates as the location changes
- 🎯 Map auto-centers to current user position
- 💻 Implemented in a single HTML file

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Google Maps JavaScript API
- Geolocation API

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open `index.html` in any browser
3. Allow location permission when prompted
4. Observe live tracking on the map

---

## 🔑 Setup Google Maps API Key

To run this project, you need your own API key from the Google Cloud platform:

1. Go to **:contentReference[oaicite:1]{index=1}**
2. Create a new project
3. Enable **:contentReference[oaicite:2]{index=2}**
4. Generate an API key
5. Replace `YOUR_API_KEY` in the HTML file with your key
